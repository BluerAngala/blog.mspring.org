主观题冲刺46：无论你犯了多少错，或者进步得有多慢，你都走在了那些不曾尝试的人的前面。

CH · 2019-11-26 · 次阅读

今日法考

  • 无论你犯了多少错，或者进步得有多慢，你都走在了那些不曾尝试的人的前面。

每天一题

基本案情：

材料一:2017年2月，甲公司与乙公司订立书面买卖合同约定：“甲将一辆汽车出卖给乙，乙支什价款100万元。2017年3月15日，甲公司交付汽车。2017年5月15日，乙公司付款。若乙公司到期未支付价款，乙公司的法定代表人负贵偿还。“甲公司法定代表人张三在合同上签字，并加盖甲公司印章；乙公司法定代表人李四也在合同上签字，但未加盖乙公司印章。2017年3月1日，乙公司法定代理人由李四变更为蔡大伟。
2017年3月15日，甲公司将汽车交付给乙公司，并办理了过户登记手续。2017年5月15日，乙公司却未支付价敖。甲公司向乙公司多次催要价款无果后，于2017年7月，将乙公司、蔡大伟告上法庭，请求乙公司支付价款，并请求蔡大伟承担连带清偿责任。乙公司辩称：心该合同是乙公司原法定代表人李四以个人名义答订的，乙公司并未盖章，故该合同对乙公司没有约束力；＠）甲公司是一家从事商品零售的公司，出卖汽车超越经营范围，该合同应属无效。蔡大伟辩称：该合同是乙公司原法定代表人李四以个人名义签订的，自己并未答字，故该合同对自己没有约束力。
材料二：在法院协助下，甲公司与乙公司达成协议，约定：＂乙公司于2017年10月15日，向甲公司支付汽车价款。”为了担保乙公司价金债务的履行，乙公司以房屋A向甲公司设立抵押，丙公司也以房屋B向甲公司设立抵押，均办理了抵押登记手续，但未约定承担担保责任的范围和方式。此外，乙公司的股东王五、赵六与甲公司订立保证合同，也未约定承担保证贵任的期间、范围和方式。
及至2017年8月15日，乙公司仍未支付购车款项，2017年12月15日，甲公司向乙公司、李四、丙公司、王五、赵六发出函件，要求其承担各自的贵任。
李四辩称：自己早巳不再担任乙公司的法定代表人，故不应承担付款责任。乙公司辩称：甲公司交付的汽车质量不符合约定，乙公司收到货物后即向甲公司提出质量瑕疵异议，甲公司至今未予解决。乙公司出示了质量瑕疵证明。丙公司辩称：丙公司向甲公司抵押的房屋B,其实是属于丙公司的股东丁公司所有，只不过登记在丙公司名下。王五辩称：由于甲公司向乙公司交付的汽车质量不符合约定，故拒绝承担保证责任。赵六未置可否。
材料三：2017年9月1日，甲公司对所交付的汽车进行维修，达到了约定的质量标准。2017年10月1日，经甲公司和王五、赵六的书面同意，乙公司将100万元价金债务，转让给甲公司的股东海天公司。同时，海天公司以房屋C向甲公司提供抵押，办理了抵押登记手续。2017年11月1日，甲公司放弃了海天公司所提供的房屋C上的抵押权。

根据材料一，请回答如下问题：（共6分）

l乙公司能否以“乙公司并未盖章”为由，拒绝承担付款义务？为什么？(2分）
2乙公司能否以“甲公司超越经营范围”为由，拒绝承担付款义务？为什么？(2分）
3蔡大伟能否以“自己并未签字”为由，拒绝承担付款义务？为什么？(2分）
答案解析
1.不能。(1分）法定代表人所实施的，以为法人设立权利、负担义务为目的的行为，系代表行为，法人要承担该行为的法律后果。(1分）
2.不能。(1分）法人超越经营范围订立的合同，未违反法律、行政法规关于特许、限制经营行业的强制性规定的，该合同有效。(1分）
3.可以。(1分）“法定代表人负责偿还”的约定，是乙公司原法定代表人李四的个人行为，由其个人承担后果。现代表人蔡大伟并未签字，不承担李四个人行为的法律后果。(1分）

  • 法定代表人的个人行为与代表行为
  • 法人超越经营范围订立合同的效力

CH总结

  • 无权代理：【狭义无权代理（合同无效）】+【表见代理（合同效力待定）】
  • 合同效力待定：经追认，合同有效，约束合同双方；拒绝追认，合同对被代理人无效（不承担责任），由实际行为人承担合同责任

根据材料二，请回答如下问题：（共12分）

4.李四能否以“巳不再担任乙公司法定代表人”为由，拒绝承担付款义务？为什么？(2分）
5李四在甲公司与乙公司的买卖合同中，居于怎样的法律地位？为什么？(2分）
6.乙公司能否以“汽车质量不符合约定”为由，拒绝承担付款义务？为什么？(2分）
7.丙公司能否以“房屋B属于丁公司所有“为由，拒绝承担担保责任？为什么？(2分）
8.王五能否以“汽车质量不符合约定”为由，拒绝承担保证责任？为什么？(2分）
9设：赵六向甲公司清偿了100万元后，可否向乙公司追偿？为什么？(2分）
答案解析
4.不可以。(1分）“法定代表人员负责偿还”的约定，是乙公司原法定代表人李四的个人行为，由李四个人承担后果，与李四是否仍为法定代表人无关。(1分）
5.保证人。(1分）“法定代表人负责偿还”的约定，以“乙公司不履行债务”为前提，为保证允诺。(1分）
6.可以。(1分）在甲公司先交货，乙公司后付款的请况下，甲公司交货质量不符合约定时，乙公司可以根据先履行抗辩权拒绝付款。(1分）
7.不可以。(1分）因房屋B登记在丙公司名下，甲公司可以相信该房屋是丙公司的，进而可以善意取得该房屋的抵押权。(1分）
8.可以。(1分）债务人对债权人享有抗辩权的，保证人可以继续以之抗辩债权人。(1分）
9.不可以。(1分）债务人对债权人享有抗辩权，保证人未以之抗辩债权人，而承担了保证责任的，不得向债务人追偿。(1分）

  • 法定代表人的个人行为与代表行为
  • 保证允诺
  • 先履行抗辩权
  • 善意取得
  • 保证人的债务人抗辩权
  • 保证人的追偿权

CH总结

  • 01.债务承担：转移的债务承担（全部转）+并存的债务承担（一起抗）
  • 02.债务承担具有无因性，自从承担之后，所谓的原因消灭，不影响债务承担。
  • 03.对于xx获得了利益，分情况讨论：有债权获利，合法；无债权获利，不当得利（应当予以返还）
  • 04.未约定情况下，买方（发现品质瑕疵比较迟），买方承认欠卖方钱或者用了商品，不代表放弃了品质瑕疵异议（用了还是可以说不好！）
  • 05.债权、质权转让转移，未通知债务人，此后，债务人向原债权人履行义务的，是履行对象正确，债权债务消灭。（合同相对性）
  • 06.抗辩权的延续：债权、债务无论转让多少次，最后拥有该债权、债务的人都可以基于原债权、债务存在的抗辩权，进行抗辩。

根据材料三，请回答如下问题：（共6分）

10.设：2017年9月10日，赵六向甲公司清偿了100万元后，可否请求王五分担50万元？为什么？(2分）
11.2017年10月10日，甲公司可以对谁主张担保权？为什么？(2分）
12.2017年11月10日，甲公司可以对谁主张担保权？为什么？(2分）
答案解析
10.不可以。(1分）连带共同保证人承担保证责任后，应先向债务人追偿。不能追偿的部分，才能够请求其他共同保证人分担。(1分）
11.主债务转让，未经第三担保人同商的，担保责任免除。(1分）故此时，甲公司可以对王五、赵六主张保证权，并可对乙公司的房屋A、海天公司的房屋C主张抵押权。(1分）
12.债权人放弃债务人提供的物保，第三担保人在弃权范围内，担保责任免除。(1分）故此时，在弃权范围之外，甲公司可以对王五、赵六主张保证权，并可对乙公司的房屋A主张抵押权。(1分）

  • 连带共同保证人的追偿权应分担请求权
  • 主债务转让对担保责任的影响
  • 共同担保中的弃权与免责

CH总结

  • 01.合同效力待定：经追认，合同有效，约束合同双方；拒绝追认，合同对被代理人无效（不承担责任），由实际行为人承担合同责任
  • 02.合同抗辩权：【先履行抗辩权】+【不安抗辩权】+【同时履行抗辩权】
  • 03.关于撤销权：
  • 【可撤销合同：诉讼或者仲裁】
  • 【效力待定合同（限制行为能力人+狭义无权代理）：单方通知】
  • 【债的保全（债权人撤销）：只能诉讼】
  • 【赠予合同（法定解除权+任意解除权）：单方通知】
  • 04.债的保全（债权人撤销），对于第三人不等价（廉价）处分财产，应当以第三人恶意为条件。
